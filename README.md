## Description

Simple API for a chat app. Supports multiple conversations for several users.
No auth yet.

## Installation

```bash
$ yarn install
```

## Running the app

Firstly, set the environmental variables (see .env.example).

### Local

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

### Docker

```bash
$ docker-compose up -d
```
